# 项目总结
## 项目背景

```
根据《天津市津南区创建智慧城市重点项目信息表》中第十七项“智慧社区”的建设内容与考核指标要求，津南区民政局开展了津南区智慧社区升级改造项目，计划以辛庄镇三鑫社区为试点社区进行建设实施。
该项目用于服务中老年群众，方便政府收集群众信息及组织居民活动。

```
## 需求分析
* 为居民服务
```
通过社区服务平台，多渠道获取居民心声，了解居民需求，更好地解决居民的难题。通过信息化手段，搭建社区服务供需对接桥梁，为居民提供更多服务选择，居民生活便利了。
```
* 为社区工作者服务
```
准确可快速地数据更新，帮助他们掌握全面、准确辖区基本信息和动态，做到“底数清、情况明”，为辖区居民提供更好服务奠定管理基础，提升社区工作者主动为居民提供服务的能力。同时极大地提升工作效率和工作质量，减轻社区工作者的工作负担。
```

```
登录注册等基础数据库建设
```

* 项目建设原则

```
1. 易操作、易维护
    用了vue框架，利用mvvm原理，实现模块分离，降低模块间的耦合度。易于操作，维护
2. 安全性原则
    系统在用户权限管理上实现统一有效的管理和认证，用了基于token的身份验证方法。
    * 客户端使用用户名跟密码请求登录
    * 服务端收到请求，去验证用户名与密码
    * 验证成功后，服务端会签发一个Token，再把这个 Token 发送给客户端
    * 客户端收到Token以后可以把它存储起来，比如放在 Cookie 
    * 客户端每次向服务端请求资源的时候需要带着服务端签发的 Token
    * 服务端收到请求，然后去验证客户端请求里面带着的 Token，如果验证成功，就向客户端返回请求的数据
    token是有时效的，一段时间之后用户需要重新验证。一定程度上增加了信息的安全性。
```

```
    用于项目打包
```

* 项目内容
```
1、服务
    nodejs
2. 登录注册
    注册：注册个人基本信息，获取信息传到服务端，存储在数据库中。
    登录：将登录的信息与数据库中信息对比，若有信息匹配者，允许登录，否则弹出信息错误，请重新登录或前往注册页面。
3、一口式受理平台建设
    对居民开放的平台。
    首先将最新活动从数据库中获取，以轮播形式展现在页面中，居民可以通过页面进行浏览及访问。
    活动信息与报名页：当点击轮播的某一页活动时，将详情通过vuex数据容器展现在活动信息栏中。报名栏：设置姓名与联系方式为必填项。将信息获取后发往服务端，在服务端进行数据存储。存储在数据库中。点击报名按钮时，判断信息不完整的情况下应提示的信息为：请填写xx。完整情况下提示：报名成功！
    人员管理页：将居民报名信息展示在此页！
4、公众服务平台建设
    信息管理：管理者在此页管理活动信息，其中栏目管理栏需要有图片的上传和替换，利用ui框架实现。添加栏目用v-for循环。
    活动管理：管理者在此页添加活动信息。可以邀请居民，同时如果是管理员身份可以对居民信息进行增加和删除。
```
